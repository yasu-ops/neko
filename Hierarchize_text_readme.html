★★テキスト階層化アプリ 使い方★★
  • テキストボックス新規作成
  • Enterキー　テキストボックスの中の文字列の末尾でEnterキーを押すと、真下に新たなテキストボックスが生成される。
  • テキストボックスの中にカーソルがあるとき
  • 上矢印キーを押すと…　上のテキストボックスへ移動。
  • 下矢印キーを押すと…　下のテキストボックスへ移動。
  • テキストボックスの前後にあるボタン
  • 消　その行のテキストボックスを削除。
  • →（またはタブキー）　テキストボックスを右にずらす。
  • ←（またはシフト＋タブキー）　テキストボックスを左にずらす。
  • ↑（またはctrl+上矢印キー）　テキストボックスを１つ上に移動。
  • ↓（またはctrl+下矢印キー）　テキストボックスを１つ下に移動。
  • たたむ　下位のテキストボックスを隠す。
  • ひらく　下位のテキストボックスを再表示。
  • G　下位のテキストボックスとグループ化。★上下移動の際、そのかたまりごと動く。★Gをもう一度押すと解除される。
  • 画面上部のボタン
  • csv保存　すべてのテキストボックスの中身を保存。
  • ファイルの選択　保存した内容をテキストボックスに読み込む。
  • 全グループ化解除　テキストボックスのグループ化を解除。
  • 画面下部のボタン
  • word文書として保存　すべてのテキストボックスの中身を１つのword文書にまとめて保存。
  • 左→右のツリーボタン　テキストボックスの階層構造をもとに左右のツリーを生成。
  • 上→下のツリーボタン　テキストボックスの階層構造をもとに上下のツリーを生成。
  • ツリー図で分岐した枝を再び１本に戻したい場合は…　分岐した枝にあたる、（並列された状態にある）各テキストボックスのうち、先頭のテキストボックス以外のテキストボックスについて、行頭に半角の+を入れる。そのあとに下位のテキストボックスを追加すると、並列されたテキストボックスが全部、その下位のテキストボックスにつながり、分かれた枝が１本に戻る。
  • マインドマップ　マインドマップを生成　※注意　マインドマップを作るためには（１）最初のテキストボックスは字下げせず、（２）２つ目以降のテキストボックスがすべて字下げされている必要がある。
